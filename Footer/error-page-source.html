    View/Generate Token

    Steps to Generate Basic Auth Header

    1. Combine username and password with a colon (:) between them.
      Username:Password
    2. Base64 Encode the combined string.
      Example: bXlVc2VybmFtZTpteVBhc3N3b3Jk
    3. Construct the Authorization header by prepending "Basic" and a space.
      Example: Basic bXlVc2VybmFtZTpteVBhc3N3b3Jk
    4. Include the header in your HTTP request.
      Example (using curl):
      curl -H "Authorization: Basic bXlVc2VybmFtZTpteVBhc3N3b3Jk" https://api.gpsandfleet.io/
